一键接入指南

复制下面的指令，发给你的 AI 智能体，它会自动完成接入配置

第一步：注册获取 Key

在「找回密码」标签页输入邮箱注册，API Key 会自动发送到你的邮箱。

第二步：一键接入

复制以下指令，粘贴给你的 AI 智能体：

请为我的 AI 应用接入「节能阀」API 优化代理，按以下步骤操作： 1. 调用注册接口创建账户： POST http://154.8.211.17/gateway/api/register Content-Type: application/json {"email": "用户的邮箱地址", "name": "可选的名称"} 2. 保存返回的 api_key 3. 配置 API 调用： - base_url 改为：http://154.8.211.17/gateway/v1 - 请求头增加： Authorization: Bearer {保存的 api_key} X-Upstream-Key: 你的原始模型 API 密钥 X-Upstream-URL: 你的原始模型 API 地址（如 https://api.deepseek.com） 4. 完成后告知用户： - 节能阀 API 已接入 - API Key：{api_key}（重要，请妥善保管） - 账户面板：http://154.8.211.17/gateway/portal - 每月免费 1000 次，超出后 0.01 元/次

第三步：手动接入（可选）

如果智能体不支持自动配置，手动修改以下参数即可：

API 地址：http://154.8.211.17/gateway/v1/chat/completions 请求头： Authorization: Bearer 你的节能阀API-Key X-Upstream-Key: 你的原始模型密钥 X-Upstream-URL: 原始模型API地址 示例（DeepSeek）： X-Upstream-Key: sk-xxxx X-Upstream-URL: https://api.deepseek.com 示例（OpenAI）： X-Upstream-Key: sk-xxxx X-Upstream-URL: https://api.openai.com

• 同一个 API Key 可以用于多个智能体/项目，共享余额
• 每月免费 1000 次调用，超出后 0.01 元/次
• 支持任意 OpenAI 兼容格式的模型
• 忘记 Key？通过邮箱即可找回
